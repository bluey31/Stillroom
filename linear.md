# Stillroom — Linear theme
#
# Linear's Interface theme is just two colors: an accent and a background.
# Set Interface theme to Custom, then paste the two values below.

# ──────────────────────────────────────────────────────────────────
# Accent     #B9B885    lichen
# Background #191714    mantle
# Contrast   30         (default)
# ──────────────────────────────────────────────────────────────────

## Install

Linear → Preferences (`G` then `S`) → Preferences → Interface theme → Custom

- Accent: paste `#B9B885`
- Background: paste `#191714`
- Leave Contrast at 30 (default)

## Why these two colors

**Background — mantle (`#191714`), not base (`#211e1c`).**
Linear's UI stacks many cards (issue rows, sidebar sections, lists). With mantle as the page background, Linear's own card surfaces sit slightly raised from the page, giving depth without needing borders. Using base would flatten everything to one plane.
